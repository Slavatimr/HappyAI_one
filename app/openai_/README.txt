Папка `openai` содержит модули, ответственные за взаимодействие с API OpenAI для обработки запросов пользователей и создания ассистентов и потоков.

Основные функции включают:
- Преобразование речи в текст и обратно.
- Обработка текстовых запросов и получение ответов от ассистентов.
- Создание новых ассистентов и потоков для ведения диалогов.

Эти модули обеспечивают интеграцию с OpenAI, позволяя приложению выполнять сложные задачи обработки языка.
