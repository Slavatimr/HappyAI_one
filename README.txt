Проект Telegram Бот с OpenAI

Этот проект представляет собой Telegram бота, который использует API OpenAI.
Бот может отвечать на команды и отвечать голосовыми сообщениями на голосовые сообщения пользователя.


Структура проекта

Проект Telegram Бот с OpenAI
├── app
│   ├── config.py
│   ├── main.py
│   ├── routers
│   │   ├── command.py
│   │   └── voice.py
│   ├── file_operations
│   │   └── file_manager.py
│   ├── openai
│   │   └── openai_handler.py
│   └── _temp_files
│       └── (временные файлы)
└── fake_db
    ├── assistants.json
    └── threads.json

Описание компонентов

- app/config.py: Управляет настройками приложения, загружая токены из файла окружения.
- app/main.py: Основной модуль, инициализирующий бота и запускающий его работу.
- app/routers: Содержит обработчики маршрутов для обработки сообщений и команд.
  - app/routers/command.py: Обрабатывает текстовые команды, такие как /start.
  - app/routers/voice.py: Обрабатывает голосовые сообщения, преобразуя их в текст и обратно.
- app/file_operations/file_manager.py: Управляет загрузкой и сохранением файлов, включая работу с JSON.
- app/openai/openai_handler.py: Обрабатывает взаимодействие с API OpenAI.
- app/_temp_files: Хранит временные файлы во время работы программы.
- fake_db: Содержит файлы для имитации базы данных, такие как assistants.json и threads.json.

Установка

1. Клонируйте репозиторий:
   git clone https://github.com/Slavatimr/HappyAI_one

2. Установите необходимые зависимости:
   pip install -r requirements.txt

3. Создайте файл .env в корне проекта и добавьте токены:
   BOT_TOKEN=ваш_токен_бота
   OPENAI_TOKEN=ваш_токен_OpenAI

Запуск

Для запуска бота выполните команду из корневой директории:
python -m app.main

Использование

- Отправьте команду /start в чат с ботом, чтобы начать взаимодействие.
- Отправьте голосовое сообщение, и бот преобразует его в текст, обработает и ответит голосом.
